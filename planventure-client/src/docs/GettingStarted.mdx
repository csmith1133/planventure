# Getting Started

## Installation

```bash
git clone https://github.com/yourusername/planventure.git
cd planventure
npm install
```

## Quick Start Guide

1. Set up your environment variables:
```bash
cp .env.example .env
```

2. Start the development server:
```bash
npm run dev
```
